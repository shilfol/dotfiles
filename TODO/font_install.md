## Rictyのインストール

### Download

#### Migu 1M
http://mix-mplus-ipa.sourceforge.jp/migu/

#### Inconsolata,Script
http://www.rs.tus.ac.jp/yyusa/ricty.html

~~~
apt-get install fontforge  

cp Inconsolata.otf ~/.fonts  

unzip migu~~~~.zip  
cp migu~~~/\*.ttf ~/.fonts

sh ricty\_generator.sh auto  

cp \*.ttf ~/.fonts
~~~

フォントをターミナルにて設定
